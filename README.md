# Letters Eater

A small Python script that removes vowels from a word.

## What it does
- Prompts for a word
- Converts it to uppercase
- Prints only non-vowel characters (`A, E, I, O, U` are skipped)

## Run
On Windows (from this folder):

```powershell
python .\lettersEater.py
```

## Example
Input:
`Developer`

Output:
`D`
`V`
`L`
`P`
`R`
